# Organização de Computadores e Linguagens de Montagem II — Labs

Repositório com as práticas desenvolvidas na disciplina Organização de Computadores e Linguagens de Montagem II,
ministrada na Universidade Federal do Ceará — Campus Quixadá.

---

# Objetivo

Este repositório tem como finalidade organizar e documentar todas as práticas da disciplina, servindo como material de estudo e portfólio acadêmico.

Os experimentos envolvem:

- Programação em baixo nível
- Manipulação direta de registradores
- Desenvolvimento bare-metal
- Uso de Makefile
- Controle de periféricos (GPIO)

---

# Como compilar

Dentro da pasta da prática:

```
make
```

 Limpar build
```
make clean
```
